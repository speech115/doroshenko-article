# Статья — Матвей Дорошенко

Сайт: **https://doroshenko-article.vercel.app**

## Как редактировать

1. Открой файл `index.html` в этом репозитории
2. Нажми иконку карандаша (✏️) справа сверху
3. Внеси изменения прямо в браузере
4. Нажми зелёную кнопку **"Commit changes"** внизу
5. Через ~30 секунд изменения появятся на сайте

## Что где находится в index.html

Весь сайт — это один файл `index.html`. Внутри:

- **Тексты** — ищи русский текст между тегами `<p>`, `<h2>`, `<span>`
- **Ссылки на форму** — ищи `href="https://forms.yandex.ru/"` и замени на свою ссылку
- **Фото** — файл `matvey.jpg` в этой же папке
- **Видео-отзывы** — файлы `review-*.mp4`
- **Скриншоты** — файлы `proof-*.png` / `proof-*.jpg`

## Как заменить картинку или видео

1. Нажми **"Add file"** → **"Upload files"**
2. Загрузи новый файл с **тем же именем** (например `matvey.jpg`)
3. Commit — файл заменится и сайт обновится

## Как поменять ссылку на форму

Найди в `index.html` все места с `forms.yandex.ru` и замени на свою ссылку.
